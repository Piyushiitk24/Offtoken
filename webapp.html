🏠 Local Setup for Maximum Privacy

For enhanced privacy, security, and offline use, run TokenForge locally on your machine:

# Clone the repository
git clone https://github.com/Piyushiitk24/Offtoken.git
cd Offtoken

# One-command setup and launch
python3 setup.py
🔒 Local Benefits:
• Complete offline functionality
• No internet required after setup
• Process sensitive documents privately
• Full feature set including OCR for scanned PDFs
• Faster processing with Streamlit interface
